# BlackJack online game documentation

## Description
Blackjack Online is a full-stack multiplayer Blackjack platform built as a hands-on learning project covering the modern Java ecosystem. The backend is powered by Spring Boot 3, featuring a RESTful game API, JWT-based authentication via Spring Security, real-time multiplayer through WebSockets with STOMP, and persistent game history and player stats backed by PostgreSQL with Spring Data JPA. The frontend is built in Angular 17, consuming the REST API and WebSocket channels to deliver a live, interactive game experience.
Card shuffling is handled through an external deck API, and schema versioning is managed with Flyway migrations. The entire stack is containerised with Docker and deployed to a Hetzner VPS, served behind an Nginx reverse proxy with SSL, and automated through a GitHub Actions CI/CD pipeline.
